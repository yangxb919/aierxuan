## Target Featured Snippet Opportunities

**Question:** "What documents do I need to import laptops from China to USA?"

**Current answer:** Generic trade documents list

**Our answer opportunity:** Complete checklist with verification steps:
- Commercial Invoice (with HS Code 8471.30)
- Bill of Lading / Air Waybill
- FCC Declaration of Conformity (with FCC ID verification)
- UN38.3 Test Summary + Test Report
- Packing List (detailed battery specs)
- ISF Filing Confirmation
- Importer Security Filing (10+2)

**Question:** "How much is customs duty on laptops from China?"

**Current answer:** "0% or varies" (incomplete)

**Our answer opportunity:**
- Base duty: 0% (HS 8471.30.01)
- MPF: 0.3464% (max $538.40)
- HMF: $26.22-$538 depending on entry type
- Section 301 tariffs: 7.5%-25% (2025 update)
- Total effective rate: 8.2%-26% of customs value
