- 日期：2026-05-08 15:32

## 处理

10 张 redesign 首屏图（home/products/oem/about/blog/faq/contact hero + 3 个产品族图）：

```
PNG 1.5-2.0 MB ──cwebp -q 82 -m 6──> WebP 64-136 KB
合计 17 MB → 972 KB（节省 94%）
```

代码引用更新：
- `redesign/TechPrimitives.tsx` redesignImages → .webp
- `lib/seo.ts` DEFAULT_OG_IMAGE → home-hero.webp
- `features/BlogListClient.tsx` BLOG_CARD_FALLBACK_IMAGE → blog-hero.webp

git rm 删除 10 张 PNG 减小仓库。

## VPS 配置补丁

发现 nginx 静态资源正则没包含 webp，导致 webp 走主 location 拿到只有 120s 的 cache-control。修复：

```diff
- location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2|ttf|eot)$
+ location ~* \.(jpg|jpeg|png|gif|webp|avif|ico|css|js|mjs|svg|woff|woff2|ttf|eot)$
```

reload 后验证 webp 拿到 `cache-control: public, max-age=31536000, immutable`。
