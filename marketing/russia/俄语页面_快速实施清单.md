## 🆘 遇到问题？

### 问题1：翻译文件不生效

```
检查：
1. 文件路径是否正确？locales/ru/common.json
2. JSON格式是否正确？（用 jsonlint.com 验证）
3. 是否重启了开发服务器？
```

### 问题2：路由 /ru 无法访问

```
检查：
1. next.config.js 是否配置正确？
2. 是否重启了服务器？
3. 浏览器缓存是否清除？
```

### 问题3：HTML源代码还是英文

```
检查：
1. 是否使用了服务器端渲染？
2. 是否正确导入了翻译文件？
3. 使用 curl 检查而不是浏览器
```
