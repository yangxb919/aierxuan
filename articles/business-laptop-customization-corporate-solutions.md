### Hardware Security Options

| Feature | Description | Recommended For |
| :--- | :--- | :--- |
| **TPM 2.0** | Hardware encryption key storage | All enterprise laptops |
| **Intel vPro** | Remote management, hardware-based security | IT-managed fleets |
| **AMD PRO** | AMD's enterprise management platform | AMD-based deployments |
| **Fingerprint Reader** | Biometric authentication | All users |
| **IR Camera** | Windows Hello facial recognition | Executive/secure roles |
| **Smart Card Reader** | CAC/PIV card authentication | Government, defense |
| **Privacy Screen** | Built-in viewing angle filter | Finance, healthcare, legal |
| **Kensington Lock Slot** | Physical theft deterrent | Shared workspaces |

### BIOS/Firmware Security

| Feature | Function | Implementation |
| :--- | :--- | :--- |
| Secure Boot | Prevents unauthorized OS loading | Enable by default |
| BIOS Password | Prevents unauthorized BIOS changes | Set master password |
| Boot Order Lock | Prevents USB boot attacks | Lock to internal SSD |
| Intel Boot Guard | Verifies firmware integrity | Factory-enabled |
| AMD Platform Secure Boot | AMD equivalent | Factory-enabled |

| DLP | Microsoft Purview, Symantec DLP | Per-seat |

## FAQ

### What's the minimum order for custom business laptops?

Most OEM partners require 100-250 units minimum for basic customization (branding, pre-configuration). Full hardware customization typically requires 500-1,000 units.

### How long does enterprise customization take?

*   **Basic customization** (branding + software): 4-6 weeks
*   **Security configuration**: 6-8 weeks
*   **Full custom hardware**: 12-16 weeks

![Timeline visualization of enterprise laptop customization phases: Basic customization taking 4-6 weeks, Security configuration 6-8 weeks, and Full custom hardware 12-16 weeks](https://res.cloudinary.com/dv1cajx98/image/upload/f_auto,q_auto:eco/v1768483100/6968e919e171b93bedca3238/x1y.jpg)

### Can we test before committing to a large order?

Yes. Most enterprise programs include a pilot phase of 10-50 units for validation before production orders.

### What happens at end-of-life?

Options include:

*   **Buyback**: Manufacturer purchases back at residual value
*   **Recycling**: Certified e-waste disposal with data destruction certificate
*   **Donation**: Refurbishment for charitable donation (tax benefits)

### How do we handle international deployments?

Custom business laptop programs can include:

*   Region-specific keyboard layouts
*   Local power adapters
*   Country-specific certifications
*   Multi-language OS images
*   Regional warranty coverage

### What security certifications are available?

| Certification | Description | Industries |
| :--- | :--- | :--- |
| FIPS 140-2 | Cryptographic module validation | Government, defense |

| TCO Certified | Sustainability + security | All |
