## How to Add These FAQs

1. Log in to the admin panel: `/admin/login`
2. Navigate to FAQ management: `/admin/faq`
3. Click "Add New FAQ"
4. For each FAQ:
   - Enter the English question and answer
   - Add translations for other languages (zh-CN, ru, ja, fr, pt)
   - Set the order/priority
   - Mark as "published"
   - Save
