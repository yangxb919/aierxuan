## 🚀 上传步骤 / Upload Steps

### Step 1: 打开产品文件
选择要上传的产品，打开对应的 `.md` 文件

### Step 2: 访问管理后台
导航到: `/admin/products/new`

### Step 3: 填写基本信息
从文件的 "Basic Information" 部分复制:
- **Slug**: 例如 `du61-i9-11900h`
- **Category**: 选择 "商务本"
- **Status**: 选择 "active"
- **Featured**: 勾选（如果文件中标记为 ⭐）
- **Sort Order**: 填写文件中的数字
- **MOQ**: 填写 100
- **Price**: 留空（显示"联系询价"）

### Step 4: 上传图片
为产品上传4-6张高质量图片:
- 产品正面照
- 侧面照
- 键盘/触控板特写
- 接口展示
- 使用场景照
- 可选: 包装照片

### Step 5: 填写英文内容 (Content)

1. **Title** (标题)
   - 从文件复制 "Product Title"

2. **Short Description** (简短描述)
   - 从文件复制 "Short Description"

3. **Long Description** (详细描述) ⭐⭐⭐
   - 从文件复制 "Long Description (Markdown)" 部分
   - **这是完整的Markdown格式描述，包含所有详细信息**
   - 直接粘贴到表单的长描述字段

4. **Key Specifications** (关键规格)
   - 点击 "+ Add Spec" 添加规格
   - 从 "Key Specifications (Structured Data)" 部分添加每一项

5. **SEO Title** (SEO标题)
   - 从文件复制 "SEO Title"

6. **SEO Description** (SEO描述)
   - 从文件复制 "SEO Description"

### Step 6: 多语言翻译
1. 确保英文内容已填写完整
2. 点击 **"AI Translation"** 按钮
3. 系统会自动翻译成其他5种语言:
   - Russian (俄语)
   - Japanese (日语)
   - French (法语)
   - Portuguese (葡萄牙语)
   - Chinese Simplified (简体中文)
4. 检查并调整翻译结果

### Step 7: 预览和提交
- 检查所有必填字段
- 确认图片已上传
- 检查英文内容完整性
- 点击 **"Create Product"** 提交

## ❓ 常见问题 / FAQ

**Q: Long Description在哪里？**
A: 每个产品的独立 `.md` 文件中都有 "## Long Description (Markdown)" 部分，包含完整的详细描述。

**Q: 需要手动翻译吗？**
A: 不需要！填写完英文内容后，点击 "AI Translation" 按钮即可自动翻译。

**Q: 图片从哪里获取？**
A: 需要准备产品的实际照片，可以本地上传或通过URL添加。

**Q: 价格怎么填写？**
A: 留空即可，系统会显示 "Contact for Price"（联系询价）。

**Q: 必须填写所有语言吗？**
A: 不需要，只有英文是必填的。其他语言可以使用AI翻译或手动填写。
