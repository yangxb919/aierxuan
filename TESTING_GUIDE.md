## 🐛 常见问题排查

### 问题 1: 组件不显示
**检查**:
- 组件是否正确导入
- 是否有 JavaScript 错误
- Supabase 连接是否正常

### 问题 2: 样式错误
**检查**:
- Tailwind CSS 是否正确配置
- 类名是否拼写正确
- 是否有 CSS 冲突

### 问题 3: 多语言不工作
**检查**:
- 语言切换器是否正常
- 翻译对象是否完整
- useLanguage hook 是否正常

### 问题 4: 表单提交失败
**检查**:
- Supabase 配置是否正确
- 表单字段名是否匹配
- 网络请求是否成功

### 问题 5: 图片不显示
**检查**:
- 图片路径是否正确
- 图片是否存在
- Next.js Image 组件配置
