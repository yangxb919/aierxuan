## 原筛选报告中需要收敛的表达

1. 不要过度承诺 AI 功能。
   - 未确认具体 SKU 具备 NPU、Copilot+ 或本地 AI 算力前，不要在标题或产品页中直接承诺这些能力。
   - 建议表达：`AI-ready configuration options`，并注明性能取决于芯片和项目要求。

2. 不要把 Snapdragon、RTX 50、OLED 当作默认产品能力。
   - 这些词可以作为 Blog 趋势分析和配置选项出现。
   - 只有内部确认有真实 SKU 和供应能力后，才能作为产品页主卖点。

3. 不要使用模糊“超长续航”。
   - 必须写电池容量、测试条件、亮度、播放/办公场景、系统配置。
   - 示例口径：本地 1080p 视频连续播放条件下达到 X 小时。

4. 游戏本不能只写 GPU 型号。
   - 必须补充 TGP、散热结构、风扇噪音、屏幕刷新率、供电和适配器信息。

5. OEM/ODM 能力不能泛泛而谈。
   - Logo、包装、BIOS、系统镜像、键盘语言、认证、交付周期都要写成可执行范围。
