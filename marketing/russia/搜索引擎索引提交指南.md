# 一、Yandex Webmaster 提交（俄罗斯市场必做）

## 1.1 注册/登录 Yandex Webmaster

**地址**: https://webmaster.yandex.com/

使用您已有的 Yandex ID 登录（与 Yandex Direct 相同的账号）

---

## 1.2 添加网站

1. 点击 **"+ Add site"** 或 **"+ Добавить сайт"**
2. 输入网站地址：`https://aierxuanlaptop.com`
3. 点击 **"Add"**

---

## 1.3 验证网站所有权

Yandex 提供多种验证方式，推荐使用 **Meta 标签** 或 **DNS 验证**：

### 方式 A：Meta 标签验证（最简单）

1. Yandex 会给您一个类似这样的代码：
```html
<meta name="yandex-verification" content="xxxxxxxxxxxxxxxx" />
```

2. 将此代码添加到网站的 `<head>` 中

**修改文件**: `src/app/layout.tsx` 或创建专门的验证组件

```tsx
// 在 layout.tsx 的 metadata 中添加
export const metadata: Metadata = {
  // ... 其他配置
  verification: {
    yandex: 'xxxxxxxxxxxxxxxx', // 替换为实际的验证码
  },
}
```

3. 部署网站后，回到 Yandex Webmaster 点击 **"Check"**

### 方式 B：DNS 验证

1. 在您的域名 DNS 设置中添加 TXT 记录
2. 记录值由 Yandex 提供
3. 等待 DNS 生效（通常 5-30 分钟）

---

## 1.4 提交 Sitemap

验证成功后：

1. 进入网站仪表板
2. 点击左侧菜单 **"Indexing"** → **"Sitemap files"**
3. 点击 **"Add"**
4. 输入：`https://aierxuanlaptop.com/sitemap.xml`
5. 点击 **"Add"**

---

## 1.5 配置区域定向（重要！）

由于您主要针对俄罗斯市场：

1. 进入 **"Indexing"** → **"Regional settings"**
2. 为 `/ru/*` 路径设置区域为 **Russia（Россия）**
3. 这有助于俄语页面在俄罗斯用户搜索时获得更好排名

---

## 1.6 Yandex Webmaster 其他有用功能

| 功能 | 说明 |
|------|------|
| **Search queries** | 查看用户通过哪些关键词找到您的网站 |
| **Indexing** | 监控页面索引状态 |
| **Site diagnostics** | 检查技术问题（404、重定向等） |
| **Turbo pages** | 俄罗斯用户的快速加载页面（可选） |
| **Original texts** | 保护原创内容 |

# 二、Google Search Console 提交

## 2.1 注册/登录 Google Search Console

**地址**: https://search.google.com/search-console

使用 Google 账号登录

---

## 2.2 添加网站

1. 点击左上角下拉菜单 → **"Add property"**
2. 选择 **"URL prefix"** 方式
3. 输入：`https://aierxuanlaptop.com`
4. 点击 **"Continue"**

---

## 2.3 验证网站所有权

推荐使用 **HTML 标签** 验证：

1. Google 会给您一个类似这样的代码：
```html
<meta name="google-site-verification" content="xxxxxxxxxxxxxxxx" />
```

2. 添加到 `src/app/layout.tsx` 的 metadata 中：

```tsx
export const metadata: Metadata = {
  // ... 其他配置
  verification: {
    google: 'xxxxxxxxxxxxxxxx', // 替换为实际的验证码
    yandex: 'xxxxxxxxxxxxxxxx', // Yandex 的验证码
  },
}
```

3. 部署后点击 **"Verify"**

---

## 2.4 提交 Sitemap

1. 在左侧菜单点击 **"Sitemaps"**
2. 在 "Add a new sitemap" 输入框中输入：`sitemap.xml`
3. 点击 **"Submit"**

---

## 2.5 请求索引特定页面（可选）

如果希望某个页面快速被收录：

1. 在顶部搜索框输入页面 URL，如：`https://aierxuanlaptop.com/ru/oem`
2. 点击 **"Request indexing"**

---

# 三、验证代码整合（Claude Code 任务）

将以下修改添加到 Claude Code 执行计划：

**文件**: `src/app/layout.tsx`

在 `metadata` 导出中添加验证配置：

```tsx
export const metadata: Metadata = {
  // ... 现有配置保持不变

  verification: {
    google: 'YOUR_GOOGLE_VERIFICATION_CODE',  // 从 Google Search Console 获取
    yandex: 'YOUR_YANDEX_VERIFICATION_CODE',  // 从 Yandex Webmaster 获取
  },
}
```

**注意**: 验证码需要您登录各平台后才能获取，每个网站的验证码都是唯一的。
