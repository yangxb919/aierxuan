## 📖 如何使用这些文件

### 方法一：逐个产品上传

1. **打开产品文件** - 例如 `du61-i9-11900h.md`
2. **阅读文件内容** - 所有信息都已格式化好
3. **访问管理后台** - `/admin/products/new`
4. **按照文件内容填写表单**：
   - 基本信息 → 复制粘贴
   - 产品标题 → 复制粘贴
   - 简短描述 → 复制粘贴
   - **Long Description** → 从 "## Long Description (Markdown)" 部分复制完整内容
   - 关键规格 → 逐项添加
   - SEO信息 → 复制粘贴
5. **上传图片** - 4-6张产品照片
6. **使用AI翻译** - 自动翻译成其他语言
7. **提交** - 创建产品

### 方法二：查看完整指南

打开 **`UPLOAD-GUIDE.md`** 获取详细的分步说明。
