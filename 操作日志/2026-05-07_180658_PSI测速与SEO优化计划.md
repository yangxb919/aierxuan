## 关键发现

1. **RU SEO 比 EN 低 16 分**（俄语 metadata + 英文正文，搜索引擎判定语言不一致）
2. **RU Mobile LCP 10.3s 灾难级**（4G 网络下要 10 秒才看到主内容）
3. **PSI 报告 RU 页面有 console error**（待排查）
4. **意外发现 Microsoft Clarity（scripts.clarity.ms）**与 Yandex Metrica + GTM 共存，来源待确认
5. **静态资源 cache TTL 仅 1 天**（应 1 年 immutable，估算节省 177 KiB 重复下载）
6. **Render blocking CSS 800ms**（chunks/f89edb58f591bf37.css 22.6 KiB）
7. **Reduce unused JavaScript Est savings 437 KiB**（最大优化空间）
